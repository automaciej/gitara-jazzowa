**Blues** jedną z podstawowych form muzycznych w jazzie.

Blues to forma 12-taktowa, podzielona na trzy części. Strukturę bluesa
dobrze pokazuje następujący schemat harmoniczny:

|   |    |  |   |    |  |   |   |  |   |   |  |   |
| - | -- |  | - | -- |  | - | - |  | - | - |  | - |
| | | I  |  | | | I  |  | | | I |  | | | I |  | | |
| | | IV |  | | | IV |  | | | I |  | | | I |  | | |
| | | V  |  | | | V  |  | | | I |  | | | I |  | | |
